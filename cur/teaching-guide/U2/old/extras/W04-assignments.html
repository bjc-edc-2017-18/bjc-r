Week 4 Homework Assignments

    Optional assignments for class discussion, silent work, or homework (for use anytime during or after the indicated lab):

    • Other Games: Maze and Asteroids
      • These videos are designed for Scratch, not Snap!, but the two are similar, and students will likely be able to develop these two apps using Snap! early in the course. (Be sure students do use Snap!)

    You may wish to specifically assign one or more of the optional "Take it Further" problems:

    • Lab 1, Conditionals:
      • Debugging the script to draw the correct picture and keep the sprite moving forever
      • Comparing and contrasting using if-else with the method that uses no conditionals
      • Resolving lab puzzles using repeat-until
      • Revisiting the dialogue and determining if the two-color drawing script can be coded using if-else
    • Lab 1, Creating Predicates:
      • Solving problems with conditionals and without
      • Rewriting the script to say what time the sprite has just reached
      • Experimenting with lists and list blocks (to preview Unit 3)
    • Lab 1, Project: Developing a block that makes plurals:
      • Building a block that takes a pronoun and verb from a foreign language as input and outputs the correct form of that verb
    • Lab 1, Plurals project, continued:
      • Extending the custom plural block to keep plural from becoming cluttered with details
      • Modifying the custom plural block to handle correctly inputs with a space (or several spaces) at the end
    • Lab 2, Script Variables:
      • Making the raise (n) to the power (b) block work for negative integer exponents
