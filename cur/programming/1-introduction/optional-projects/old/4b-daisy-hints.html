Hints For Daisy Design

    1. Create a custom block drawCircle to draw a circle, via the “Variables” menu “Make A Block” option.

      The sprite moves 1 step, turns 1 degree and repeats this 360 times to make a full cycle and draw a circle. The warp block (from the Control blocks Menu) is there to make sure that the entire enclosed code is executed at once prior to a screen refresh. In the absence of the warp block, the circle will draw slowly step by step instead of all at once.
    2. Create a custom block drawDaisy that will draw the “Daisy” pattern. In the design given, there are 24 circles, which means the drawing sprite must turn 24 times each time by 360/24= 15 degrees.

    3. Now put it all together:

      This code clears the screen of any previous drawings, sets the pen color, brings the pen to the center of the screen and draws the daisy. The hide block hides the arrow sprite so only the drawing is seen.
