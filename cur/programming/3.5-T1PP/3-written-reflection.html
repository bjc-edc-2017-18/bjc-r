Writing a Reflection

    This is designed to preview the Advanced Placement Create Task. --MF

    As a team, do each of the following:

    1. Describe the purpose of your program and the language that you used (Snap!, of course) to write it. (100 words or less)
    2. Describe how your project uses abstraction: (100 words or less)
      • Include an image of part of your program that uses abstraction.
      • Describe how this part of your program uses abstraction.
    3. Describe how your project uses a complex algorithm: (100 words or less)
      • Include an image of the most complicated algorithm in your program.
      • Describe what that algorithm does.
    4. Describe how you created your program step-by-step. What did you do first? Next? What challenges did you run into and how did you solve them? (100 words or less)
