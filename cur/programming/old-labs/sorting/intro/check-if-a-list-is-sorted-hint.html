    Here are some things to think about:
    • We want this to return either true or false. Such blocks are predicate blocks, so we need to select the predicate reporter block shape when we make the block.
    • A list is in order if every sequential pair is in the correct order.
    • We can assume that the list is in order until we find evidence that it is not.
    • We are going to need an index variable to step through each pair of elements in the array.
    • It is helpful to create variables (and give names to particular useful information) to keep track of things.
