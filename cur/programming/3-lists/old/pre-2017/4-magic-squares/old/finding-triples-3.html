Listing All The Triples

    Here's one version of the lookup reporter from the previous page. It uses the higher-order function map:
    When using map, a new list is built the same size as locations. Each member of the new list is the result of the item block.
    How is this helpful? Now you can pull any set of 3 numbers from the list of 9 numbers, on demand.
    And, you have a handy list of triples you'd like to know about:
    1. Write all-triples that takes square, a set of 9 numbers, and reports a list of eight triples, the values in square for each row, column, and diagonal. Remember that triples is a global variable that can be used anywhere.

      The output from all-triples can be a little hard to read. It's a list of 8 smaller lists, each with 3 elements.
    2. Write a description of how you could take this information and determine whether the original 9 numbers form a magic square.
